# 딴지일보 → 텔레그램 자동 전송

## 개요
딴지일보 자유게시판에서 특정 작성자(정대만mitsui)의 증시 요약 글을 자동 크롤링하여 텔레그램으로 전송하는 자동화 시스템

## 구성
- **크롤러:** `ddanzi_to_telegram.py` — 딴지일보 크롤링 + 텔레그램 봇 전송
- **자동화:** GitHub Actions (`.github/workflows/ddanzi_check.yml`)
- **리포지토리:** https://github.com/edward071007-hash/ddanzi-telegram (public)

## 텔레그램 봇 정보
- 봇 토큰: GitHub Secrets에 `TELEGRAM_BOT_TOKEN`으로 저장
- Chat ID: GitHub Secrets에 `TELEGRAM_CHAT_ID`으로 저장 (1968767262)

## 실행 스케줄 (한국시간 기준)
- 06:00~08:00 — 10분 간격 (한국증시 마감 요약)
- 16:00~18:00 — 10분 간격 (미국증시 요약)
- 그 외 시간 — 실행 안 함

## 동작 방식
1. 딴지일보 자유게시판에서 "정대만mitsui" 닉네임으로 검색
2. 마지막으로 보낸 글 번호(document_srl) 이후의 새 글만 감지
3. 각 글의 작성자를 확인하여 공지글 등 다른 작성자 글은 스킵
4. 새 글 본문을 크롤링하여 텔레그램으로 전송
5. 마지막 글 번호를 GitHub Actions cache에 저장

## 비용
- 전부 무료 (Claude API 불필요, GitHub Actions public 리포 무제한)

## 확장 가능
- `TARGET_AUTHOR` 변경/추가로 다른 작성자 글도 수집 가능
